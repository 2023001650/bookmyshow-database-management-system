# BookMyShow Database Management System

## Overview
This project demonstrates the design and implementation of a Database Management System (DBMS) for an online movie ticket booking platform (BookMyShow).

## Features
- Entity Relationship (ER) Diagram
- Database Schema Design
- SQL Table Creation
- Primary Key & Foreign Key Constraints
- Aggregate Functions
- Joins
- Nested Queries
- Set Operators

## Technologies Used
- Oracle SQL
- Database Management System (DBMS)
- ER Modeling
